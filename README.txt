The Me Test Project (check v 1.0)

Contains login method based on ddt framework (data driven tests) and unittest standart Python 3.6 library.

To run test suite: right click on test_container.py file and choose 'run'

The test suite consists from 1 Positive test and 7 Negative tests for test Log In/Log Out actions.

Tests is performing as example front to ME Admin Web Site URL.

Test Scenario Positive:

1. Log In with valid credentials to Me Admin via Chrome.
2. Check Site label have a expected Title.
3. Log Out from the site.
4. Log In with same user.
